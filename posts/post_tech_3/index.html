@GeneratedValue的意義和作用

January 20, 2026
#Java

@GeneratedValue 定義

  • JAVA 持久化 API (JPA) 中的一個annotation。
  • 指定 entity 的主鍵( primary key ) 的生成策略
  • 通常會搭配 @Id 使用
  • strategy
    • GenerationType.IDENTITY：依賴DB的 AUTO_INCREMENT 特性，通常用於MySQL、MariaDB (Oracle不支持)
    • GenerationType.SEQUECE：用DB序列對象生成主鍵，通常用於Oracle、PostgreSQL (MySQL不支持)
      • 需要設定 db sequence
      • 支援batch insert
    • GenerationType.TABLE：專門有一張表來模擬序列
    • GenerationType.UUID：讓JPA自行生成一個UUID作為值
    • GenerationType.AUTO：JPA自行決定要如何產生唯一值
